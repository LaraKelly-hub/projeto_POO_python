Controle de Funcionários com Python e OOP

Projeto desenvolvido para simular o gerenciamento de funcionários utilizando *Orientação a Objetos (OOP)* em Python.
O objetivo é aplicar conceitos de lógica de programação, criação de classes, módulos e tratamento de exceções para reforçar o controle de dados.

Funcionalidades:
- Gerenciamento de funcionários com classes e subclasses.
- Tratamento de exceções para dados inválidos.
- Organização em módulos (`funcionario`, `gerenciador_RH`, etc.).

Tecnologias:
- Python
- Orientação a Objetos
